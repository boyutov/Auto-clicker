# Auto Clicker (Java)

## Описание
Данный проект — это автокликер на Java.

Программа работает следующим образом:
- При нажатии **средней кнопки мыши (СКМ)** включается автокликер
- Автокликер выполняет клики **правой кнопкой мыши (ПКМ)**
- Повторное нажатие СКМ останавливает работу автокликера

Проект предназначен для изучения работы с:
- обработкой событий мыши
- потоками (Thread)
- низкоуровневым управлением вводом

## Технологии
- Java 17
- AWT / Robot API

## Запуск
1. Убедитесь, что установлен Java 17
2. Запустите главный класс проекта
3. Используйте СКМ для включения и выключения автокликера
